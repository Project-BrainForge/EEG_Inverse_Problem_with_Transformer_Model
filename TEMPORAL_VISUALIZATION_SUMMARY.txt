HOW IT WORKS
================================================================================

Your Data:
  all_out: (num_samples, 500, 994)
  ‚Üí 500 timepoints, 994 regions each

Processing:
  For each timepoint:
    1. Extract region values (994 regions)
    2. Map to vertices (20,484 vertices)
    3. Visualize on 3D cortex

Result:
  Multiple views showing temporal evolution!

CUSTOMIZATION
================================================================================

Number of timepoints:
  visualize_timepoints(1, 9)      % 3x3 grid
  visualize_timepoints(1, 16)     % 4x4 grid
  visualize_timepoints(1, 20)     % 4x5 grid
  visualize_timepoints(1, 25)     % 5x5 grid

Specific times:
  visualize_selected_timepoints(1, [1, 50, 100, 150, 200])

View angle:
  visualize_selected_timepoints(1, 1:50:500, [-86, 17])  % Left
  visualize_selected_timepoints(1, 1:50:500, [86, 17])   % Right
  visualize_selected_timepoints(1, 1:50:500, [0, 90])    % Top

Animation speed (edit in script):
  v.FrameRate = 30;    % Normal
  v.FrameRate = 15;    % Slower
  v.FrameRate = 60;    % Faster

================================================================================
TROUBLESHOOTING
================================================================================

Error: "Expected 3D data"
  ‚Üí Your data is (N, 994) not (N, 500, 994)
  ‚Üí Solution: Use visualize_single_prediction(1)

Want all 500 in one figure?
  ‚Üí 500 subplots is too many!
  ‚Üí Solution 1: visualize_selected_timepoints(1, 1:10:500)  % Every 10th
  ‚Üí Solution 2: visualize_all_timepoints_animation(1)       % Video

Animation takes too long?
  ‚Üí Edit script: Change v.FrameRate = 30 to 15
  ‚Üí Or: Process every 2nd frame

Out of memory?
  ‚Üí Reduce number of timepoints shown
  ‚Üí Close figures: close(gcf)

================================================================================
TIME INTERPRETATION
================================================================================

Assuming 2ms sampling (500 Hz):

  Timepoint    Time
  ---------    ----
       1       0 ms
      50       100 ms
     100       200 ms
     200       400 ms
     250       500 ms
     500       1000 ms

Adjust labels in scripts if your sampling rate is different.
