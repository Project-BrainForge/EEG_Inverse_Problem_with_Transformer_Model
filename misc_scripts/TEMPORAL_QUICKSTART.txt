================================================================================
TEMPORAL VISUALIZATION QUICK START
================================================================================

If your predictions have shape (num_samples, 500, 994), you have temporal data!
Here's how to visualize all 500 timepoints:

================================================================================
OPTION 1: GRID VIEW (Recommended for Analysis)
================================================================================

Show selected timepoints in a single figure:

  >> cd misc_scripts
  >> visualize_timepoints(1, 16)       % 4x4 grid, 16 timepoints
  >> visualize_timepoints(1, 20)       % 4x5 grid, 20 timepoints
  >> visualize_timepoints(1, 25)       % 5x5 grid, 25 timepoints

Best for: Quick overview, comparing different timepoints

================================================================================
OPTION 2: CUSTOM TIMEPOINTS (Recommended for Focused Analysis)
================================================================================

Choose specific timepoints to visualize:

  % Every 50th timepoint
  >> visualize_selected_timepoints(1, 1:50:500)

  % Specific timepoints
  >> visualize_selected_timepoints(1, [1, 100, 200, 300, 400, 500])

  % Early response (first 100 timepoints)
  >> visualize_selected_timepoints(1, 1:10:100)

  % With different view
  >> visualize_selected_timepoints(1, 1:50:500, [86, 17])  % Right view

Best for: Focused analysis, publications

================================================================================
OPTION 3: ANIMATION (Recommended for Presentations)
================================================================================

Create video showing all 500 timepoints:

  >> visualize_all_timepoints_animation(1)
  
  % Save to specific location
  >> visualize_all_timepoints_animation(1, '../results/my_video.mp4')

Best for: Presentations, seeing complete temporal evolution

================================================================================

  >> pred_data = load('../source/VEP/transformer_predictions_best_model.mat');

================================================================================
CUSTOMIZATION
================================================================================

Number of timepoints:
  visualize_timepoints(1, N)     % N = 9, 16, 20, 25, etc.

Specific times:
  visualize_selected_timepoints(1, [t1, t2, t3, ...])

Animation speed (edit in script):
  v.FrameRate = 30;    % Default: 30 fps
  v.FrameRate = 15;    % Slower
  v.FrameRate = 60;    % Faster

================================================================================
TROUBLESHOOTING
================================================================================

Error: "Expected 3D data"
  ‚Üí Your data is (N, 994) not (N, 500, 994)
  ‚Üí Use regular scripts: visualize_single_prediction(1)

Too many subplots:
  ‚Üí Don't show all 500 in one figure!
  ‚Üí Use: visualize_selected_timepoints(1, 1:10:500)  % Every 10th
  ‚Üí Or: visualize_all_timepoints_animation(1)        % Animation

Animation too slow:
  ‚Üí Reduce frame rate in script
  ‚Üí Process fewer frames

Out of memory:
  ‚Üí Reduce number of subplots
  ‚Üí Close figures: close(gcf)

================================================================================
TIME INTERPRETATION
================================================================================

Assuming 2ms sampling (500 Hz):
  Timepoint 1    = 0 ms
  Timepoint 50   = 100 ms
  Timepoint 100  = 200 ms
  Timepoint 250  = 500 ms
  Timepoint 500  = 1000 ms
