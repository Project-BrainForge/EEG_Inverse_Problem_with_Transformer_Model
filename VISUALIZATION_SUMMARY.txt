================================================================================
CUSTOMIZATION
================================================================================

To customize visualization, edit visualize_transformer_predictions.m:

Line 18: Change prediction file
  prediction_file = '../source/VEP/YOUR_FILE.mat';

Lines 24-29: Adjust visualization
  num_samples_to_show = 10;   % Show more samples
  subplot_rows = 2;           % 2 rows
  subplot_cols = 5;           % 5 columns
  face_alpha = 0.9;           % Less transparent (0-1)
  threshold = 0.2;            % Higher threshold (0-1)
  view_angle = [90, 0];       % Different camera angle

Common view angles:
  [-86, 17]  = Left lateral (default)
  [86, 17]   = Right lateral
  [0, 90]    = Top view
  [0, 0]     = Front view
  [180, 0]   = Back view
